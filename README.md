# Cash Flow PWA

A Progressive Web Application for managing and tracking your daily cash flow and spending habits.

## Overview

Cash Flow PWA is a lightweight financial management tool that helps you monitor your daily expenses and manage your finances efficiently. Built with vanilla HTML, CSS, and JavaScript, it works offline and can be installed on any device.

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Add the app to your home screen to install it as a PWA

## Installation as PWA

- **Desktop:** Click the install icon in your browser's address bar
- **Mobile:** Tap "Add to Home Screen" from your browser menu

## Browser Support

This PWA works on all modern browsers that support:

- Service Workers
- Web App Manifest
- ES6 JavaScript
